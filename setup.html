AgriJobs Setup

Agriculture Hiring Platform - Setup Instructions

1

Import Database

Open phpmyadmin, create database "agriculture_jobs" and import database.sql file

2

Update Database Credentials

Edit includes/db_connection.php with your database details

3

Set Folder Permissions

Ensure uploads/ folder has write permissions (chmod 755 on Linux)

4

Test the Website

Navigate to index.php and test with demo credentials below

  • Farmer: farmer@example.com / farmer123
  • Worker: worker@example.com / worker123
  • Admin: admin@agrijobs.com / admin123
Go to Website Read Docs
Important: Make sure MySQL is running and database.sql has been imported before accessing the website.
