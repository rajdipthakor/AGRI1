- PHP 7.4 or higher
- MySQL 5.7 or higher / MariaDB 10.3 or higher
- Apache/Nginx web server with mod_rewrite enabled

## STEP 3: DATABASE SETUP

### Method A: Using phpMyAdmin
1. Open http://localhost/phpmyadmin/
2. Click "New" to create database
3. Database name: "agriculture_jobs"
4. Collation: "utf8mb4_unicode_ci"
5. Click Create
6. Select the new database
7. Click "Import" tab
8. Choose database.sql file
9. Click Import

### Method B: Using MySQL Command Line
```bash
mysql -u root -p
CREATE DATABASE agriculture_jobs;
USE agriculture_jobs;
SOURCE /path/to/database.sql;
EXIT;
```

### Method C: Using PHpStorm or IDE
1. Right-click on Databases
2. New → Database
3. Choose MySQL
4. Enter connection details
5. Create database "agriculture_jobs"
6. Run SQL Script from database.sql file

## STEP 9: TROUBLESHOOTING

### Error: "Connection failed"
- Verify MySQL is running
- Check credentials in db_connection.php
- Check database is created with correct name

### Error: "Page not found"
- Verify all files are in correct directories
- Check file names match exactly
- Clear browser cache

### Error: "Unable to upload file"
- Check uploads/ folder permissions
- Verify folder exists
- Check PHP upload size limit in php.ini

### Session issues
- Clear browser cookies
- Check session save path
- Verify PHP sessions are enabled

### 404 on admin pages
- Check admin/ folder exists
- Verify header.php relative paths
- Check URL structure

## STEP 10: CONFIGURATION ADJUSTMENTS

### Change Site Colors

```css

    --primary-color: #28a745;  /* Change green to your color */

```

Edit config.php:
```php
define('MAX_UPLOAD_SIZE', 5242880); // 5MB in bytes
```

### Change Items Per Page
Edit config.php:
```php
define('ITEMS_PER_PAGE', 10); // Change pagination
```

## STEP 11: SECURITY SETUP

### For Production (HTTPS)
1. Install SSL Certificate
2. Update db_connection.php:
```php
session_set_cookie_params(array(
    'secure' => true,  // Change to true
    ...
));
```
3. Update .htaccess to redirect HTTP to HTTPS

### Protect Admin Folder
Create admin/.htaccess:
```apache
<FilesMatch "\.(php|php\d+)$">
    Deny from all
</FilesMatch>
```

### Database Backups
Create backup of database regularly:
```bash
mysqldump -u root -p agriculture_jobs > backup.sql
```
